目前问题:
- 自己的服务器是http协议，微信小程序需要https协议的接口，导致自己写的接口不能在正式版中使用

使用步骤:
- git clone ....
- 打开微信小程序编辑器
- 导入代码
- 运行(在编辑器可是无视https协议)
